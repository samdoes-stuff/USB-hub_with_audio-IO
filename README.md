# USB-hub_with_audio-IO
A USB hub with integrated audio input/output

#Main Components
USB Hub Based on the CoreChips SL2.1A Audio DAC and ADC based on the TI PCM2912A

#Fabrication
I used JLCPCB for fabrication. You can use the provided gerber and BOM and Pick&Place file to order one yourself

#Tweaking
I have also provided the necessary files if you need to tweak and change anything I have used EasyEDA for designing the PCB Here is the Schematic Schematic_USB-Hub-with-audio-card_2025-04-20
